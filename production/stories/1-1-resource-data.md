# Story 1-1: 资源管理系统 (F2) - 基础数据结构

## 概述
实现资源管理系统的核心数据结构，用于管理 HP、护盾、行动点(AP)和粮草(Food)这四种基础资源。该任务仅包含数据结构定义和基本属性，不包含复杂的变动逻辑或跨回合计算。

## 需求来源
- GDD: `design/gdd/resource-management-system.md`

## 验收标准 (Acceptance Criteria)
- [ ] 定义 HP、护盾、行动点(AP)、粮草的数据结构或组件
- [ ] 包含属性的初始值、最大值限制规则
- [ ] 编写并通过自动化单元测试，验证：
  - HP 最大值限制与初始值
  - 护盾的初始值为 0
  - 行动点(AP)的初始上限定义
  - 粮草的初始值和上限(150)
- [ ] 代码符合项目 GDScript 编码规范，并具有文档注释

## 架构与技术笔记
- 使用 Godot 的 Node 或 RefCounted 作为数据结构的基类。建议使用独立的 Component 设计，方便挂载到不同的 Entity（如玩家、敌人）上。
- 对于跨场景资源（如粮草），后续可能需要独立管理，但当前只需定义其结构。
- 测试框架: GdUnit4 (测试应放在 `tests/unit/resource_management/` 目录下)
