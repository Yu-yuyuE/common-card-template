# 卡牌设计文档（Cards Design）

_Updated: 2026-04-03_
_Status: Draft v3.1（图鉴解锁体系 + 卡牌特殊属性全套引入 + 十三州专属卡体系；特殊属性术语统一）_

---

## Overview

常规卡规则：

- 与武将专属卡分离
- 玩家使用任意武将时均可将任意常规卡加入卡组
- 每局结束常规卡3选1

**v3核心更新：**

1. **三层图鉴解锁体系**：初始通用卡（200+）、待开启通用卡（100+）、十三州专属卡（200+）
2. **卡牌特殊属性全套引入**：固定起始、常驻、移除、临时、运筹、X费（借鉴杀戮尖塔+航海奇闻2）
3. **X费动态卡机制**引入
4. 提升负面卡占比与数量
5. 增强”对敌施加负面”与”负面转收益”体系

---

## Player Fantasy

玩家可构筑：

- 常规稳定流
- 负面控制流（给敌人上负面）
- 负面经营流（自己吃负面换收益）
- **固定起始开局流**：固定起始卡开场即在手中，构建强力的第1回合爆发
- **常驻蓄力流**：常驻卡留在手中跨回合蓄力，下回合集中爆发
- **运筹手牌流**：手中的卡发挥被动效果，不打出也有价值

## Detailed Rules

### 1) 图鉴解锁体系（三层）

**图鉴 = 玩家在地图中可获取到的卡牌池总集合。** 所有图鉴卡均可被任意武将拾取加入卡组（不绑定武将）。

#### 1.1 初始通用图鉴（200+张）

> **解锁条件**：游戏初次开始即全部开放，无需任何通关记录。

- 全部近战攻击卡（50张）
- 全部远程攻击卡（50张）
- 基础技能卡（80张）
- 基础诅咒卡（25张，含抽到触发型/常驻牌库型/常驻手牌型三种类型的代表性卡牌）

合计目标：约 **205张**

**设计原则**：这批卡应能完整支撑游戏核心玩法，任意武将单靠初始图鉴即可完成一局游戏，不造成进度阻塞感。

---

#### 1.2 待开启通用图鉴（100+张）

> **解锁条件**：在**任意十三州地图**通关（击败该地图终点Boss）后，开启一批通用图鉴卡。

**解锁规则**：

| 通关次数    | 本次新开卡牌数量 | 累计已开卡数量 |
| ----------- | ---------------- | -------------- |
| 第1次通关   | 约25张           | 约25张         |
| 第2~4次通关 | 约15张/次        | 约25+45=70张   |
| 第5~7次通关 | 约12张/次        | 约70+36=106张  |
| 第8次通关+  | 解锁收尾残余批次 | 约110+张       |

**内容构成**：

- 高阶技能卡（高费高效卡，需有基础流派意识才能使用）：约45张
- 高阶诅咒卡（效果更强烈的三类负面卡）：约30张
- 特殊机制卡（带固定起始/常驻/临时/运筹/X费等属性的中高阶卡）：约35张

合计目标：约 **110张**

**设计原则**：

- 新开卡不应让初学者感到”之前吃亏”，而是带来”哇，还有更多玩法”的惊喜感
- 每次解锁应包含至少1张”主题感强、效果有趣”的标志性卡牌

---

#### 1.3 十三州专属图鉴（200+张）

> **解锁条件**：在**指定十三州地图**通关后解锁该州的专属图鉴卡（必须使用与该地图对应武将的生涯地图才算）。

**各州专属卡数量分配（按战役比例）**：

| 州名     | 战役记录数 | 专属卡数量 | 核心主题            |
| -------- | ---------- | ---------- | ------------------- |
| **荆州** | ~26条      | **30张**   | 水战/火攻/联盟/断粮 |
| **司州** | ~16条      | **19张**   | 关隘攻防/谋略/权变  |
| **扬州** | ~18条      | **21张**   | 水军/奇袭/濡须防线  |
| **益州** | ~14条      | **16张**   | 山地/蜀道/南蛮      |
| **冀州** | ~8条       | **10张**   | 平原决战/以弱胜强   |
| **兖州** | ~8条       | **10张**   | 中原争霸/黄巾余孽   |
| **凉州** | ~5条       | **7张**    | 沙漠/羌乱/西凉铁骑  |
| **幽州** | ~5条       | **7张**    | 雪地/远征/乌桓      |
| **徐州** | ~5条       | **7张**    | 沼泽/流亡/下邳水战  |
| **并州** | ~3条       | **5张**    | 山地边境/异族       |
| **豫州** | ~2条       | **4张**    | 黄巾/中原核心       |
| **青州** | ~2条       | **4张**    | 青州黄巾/海滨       |
| **交州** | ~0条       | **3张**    | 南蛮/远境/异域      |

合计目标：约 **143张**（首版）；扩展至 **200+张** 时各州可适当增加

**设计原则**：

- 每州专属卡应体现该州的**地理特色、历史氛围、兵种特色**
- 优先适配该州的**地形标签**（荆州=水域/森林，凉州=沙漠，幽州=雪地）
- 玩家用不同武将打不同州，会遇到不同的州专属卡出现在奖励池中

---

**总图鉴规模目标**：初始205 + 待开启110 + 十三州143 = **约460张**（首版），扩展版目标 **500+**

### 2) 卡牌特殊属性系统（全套引入）

借鉴杀戮尖塔与航海奇闻2的机制，引入以下6种卡牌特殊属性标签。每张卡可携带0~2种特殊属性。

#### 2.1 固定起始（Innate）

> **定义**：带此属性的卡牌，每场战斗开始时**保证出现在初始手牌**中。

- 每个卡组中固定起始卡数量建议不超过3张（超过时随机选3张固定起始）
- 典型用途：某些装备或武将被动技能可以使玩家在第一回合固定拿到指定的牌
- **三国化命名**：「先手」

| 示例卡牌 | 费用          | 效果                                          |
| -------- | ------------- | --------------------------------------------- |
| 先手突袭 | 0费，固定起始 | 对目标造成8点伤害。（固定起始：开场即在手中） |
| 竭诚进谏 | 1费，固定起始 | 每回合开始时额外抽1张牌，持续2回合。移除。    |

---

#### 2.2 常驻（Retain）

> **定义**：带此属性的卡牌，回合结束时**常驻在手牌中，不可选**，除非被卡的效果或武将被动技能丢弃。

- 每个回合最多可常驻3张（超过自动丢弃最老的常驻卡）
- 典型用途：跨回合蓄力的大招，或有特殊条件触发的卡
- **三国化命名**：「蓄势」

| 示例卡牌 | 费用      | 效果                                                    |
| -------- | --------- | ------------------------------------------------------- |
| 蓄势一击 | 2费，常驻 | 造成18点伤害。若在手中常驻过1回合，改为造成30点伤害。   |
| 候机而动 | 1费，常驻 | 若本回合敌方攻击了我方，造成15点直击伤害；否则抽2张牌。 |

---

#### 2.3 临时（Ethereal）

> **定义**：带此属性的卡牌，若本回合结束时仍在手牌中（未被打出），**自动移除，战斗结束也不加入卡组**（除非特殊说明）。

- 临时卡通常效果极强，强制玩家当回合打出，否则浪费
- 与”使用后移除”不同：临时是”不用就移除”，移除是”用了就移除”
- **三国化命名**：「一时之计」

| 示例卡牌 | 费用      | 效果                                                         |
| -------- | --------- | ------------------------------------------------------------ |
| 计出奇策 | 0费，临时 | 对所有敌将造成12点伤害。（一时之计：本回合不打出则自动移除） |
| 天降神兵 | 1费，临时 | 本回合所有兵种卡费用变为0。                                  |

---

#### 2.4 运筹（Passive）

> **定义**：带此属性的卡牌在手牌中即可产生持续效果，**无需打出**。

- 与诅咒卡的”常驻牌库型”区别：运筹卡是正面效果，是玩家主动保留的
- **三国化命名**：「运筹」

| 示例卡牌 | 费用      | Lv1 常驻效果                           | 打出效果                     |
| -------- | --------- | -------------------------------------- | ---------------------------- |
| 帷幄筹谋 | 1费，运筹 | 每当你打出1张卡，为我方增加1点护盾     | 立即获得6点护盾，移除        |
| 纵横捭阖 | 2费，运筹 | 每回合开始时，抽到的第一张兵种卡费用-1 | 本回合所有兵种卡费用-1，移除 |

---

#### 2.5 X费动态卡（Variable Cost）

> **定义**：费用为X（剩余所有行动点），效果与X值（剩余所有行动点）成正比。

- X最小值为0（0投入也可打出，但效果最弱）
- **三国化命名**：「量力而行」

| 示例卡牌 | 费用 | Lv1 效果                                 |
| -------- | ---- | ---------------------------------------- |
| 八阵图   | X费  | 对所有敌将造成X×5点伤害，并施加X层中毒。 |
| 兵马未动 | X费  | 抽X+2张牌，并将X张牌从手牌放回牌库顶。   |
| 粮草调度 | X费  | 恢复X×4点HP，并为卡组中的X张随机卡升级。 |

### 3) 诅咒卡核心规则

负面卡分为三种类型：

| 类型           | 触发时机                             | 持续方式                                     | 示例                              |
| -------------- | ------------------------------------ | -------------------------------------------- | --------------------------------- |
| **抽到触发型** | 被抽入手牌时立即触发，触发后正常丢弃 | 单次触发，丢弃即加入弃牌堆                   | 毒药（抽到→扣2血）                |
| **常驻牌库型** | 存在于牌库中即持续生效               | **直到被净化操作移出卡组才消除，丢弃不消除** | 韬晦（在库→最大生命-3）           |
| **常驻手牌型** | 进入手牌后持续占位                   | 需支付指定费用才能丢弃，否则常驻占用手牌位   | 泥泞（需1费丢弃，否则锁住手牌格） |

通用规则：

- 负面卡可来自事件、敌人、特定卡牌或武将被动。
- 负面卡**无法主动打出**，**不可升级**。
- 负面卡仅可通过：商店净化 / 技能净化 / 特定卡牌净化，操作结果为**移出卡组**。

### 2) 常规卡基础设计（补回）

#### 2.0 升级与离场机制（新增）

- 每种卡牌可升级1次（Lv1 → Lv2），不可重复升级。
- 升级后保留原卡身份与标签，仅提升数值或追加1条轻量效果（不改变核心定位）或-1费或使用后移除效果取消。
- 升级收益建议：
  - 伤害/护甲类：+20% ~ +35%
  - 施加层数类：+1层或延长1回合
- **升级移除规则（强制）**：若升级后"使用后移除"取消（Lv1移除→Lv2不移除），则Lv2伤害或效果**应与Lv1保持一致**（不应额外提升数值）。不移除本身即为升级收益——Lv2可在同一局战斗中重复使用，这就是升级的核心价值，无需再叠加数值增益。
- **效果文案规范（强制）**：卡牌效果文案中**禁止使用"改为"**（如"若条件满足，改为造成X点伤害"）。条件触发的效果一律用**"额外"**（叠加增益）、**"必定暴击"**（特殊触发）或**"并"**（附加效果）表述。"改为"属于内部替换逻辑，不适合作为玩家可读的效果文案。

离场机制分为两类：

A. **使用后移除（战斗移除）**

- 卡牌打出后从本场战斗**移除**（不在抽牌堆/弃牌堆中）。
- 该卡在本局游戏结束后自动回到玩家卡组。
- 用于高爆发或高功能卡，防止单场循环滥用。

B. **消耗卡（地图一次）**

- 卡牌在当前地图（M1~M5）内仅可成功使用1次。
- 一旦使用，标记为“已消耗”，本地图后续战斗不可再使用。
- 进入下一张地图时重置可用次数。

追踪要求：

- 战斗内需记录“战斗移除”状态。
- 战役层需记录“地图消耗”状态（按卡ID+地图ID）。

#### 2.1 攻击卡子类型与目标规则

- 攻击卡分为：**近战** / **远程**。
- **近战攻击卡**：
  - 默认均为0费；
  - 单体伤害>=6，群体伤害>=5；
  - 默认对目标敌将造成伤害（受护甲影响）；
  - 带”无视护甲”关键词时伤害直接扣目标HP，不经护甲吸收。
- **远程攻击卡**：
  - 单体伤害>=6，群体伤害>=5；
  - 可指定敌方可见目标；默认受护甲影响，特殊文本可无视护甲；
  - 默认不允许0费（若后续出现，需在卡面与平衡表单独标注）。

#### 2.2 常规卡通用字段

每张常规卡至少包含：

- 卡名
- 等级（Lv1/Lv2）
- 费用（数字 / X / 0）
- 类型（近战攻击/远程攻击/技能/兵种/诅咒）
- 目标规则（单个/全体/任意）
- 基础效果值（伤害/护甲/层数/抽牌）
- 离场机制（常规/使用后移除/消耗卡）
- 关键标签（近战/远程/召唤/净化/施咒等）
- **特殊属性**（可多选，留空表示无）：
  - `固定起始`：战斗开始时保证在起始手牌中
  - `常驻`：回合结束时常驻在手牌，不可选，除非被卡的效果或武将被动技能丢弃
  - `移除`：使用后会从卡组移除，战斗结束重新加入卡组
  - `临时`：回合结束若未打出则自动移除，战斗结束也不加入卡组（除非特殊说明）
  - `运筹`（常驻有被动）：在手牌中即持续生效，无需打出
  - `X费`：费用为X（剩余所有行动点），效果与X值（剩余所有行动点）成正比
- **图鉴归属**：初始通用 / 待开启通用（批次1~8）/ 十三州专属（指定州名）

- 负面卡仅可通过：商店净化 / 技能净化 / 特定卡牌净化，操作结果为**移出卡组**。

## Formulas

### 近战/远程目标合法性

`TargetLegal = (CardRangeType == Melee && Target == EnemyFrontFirst) || (CardRangeType == Ranged && Target ∈ EnemyVisibleSet)`

- `EnemyVisibleSet`：敌方当前可被选中的全部目标（各自独立计算护甲）

### 近战0费强度约束

`ZeroCostMeleeBudget = ZeroCostMeleeDamage / AvgTurnDamage`

建议安全区：`0.10 ~ 0.18`

### 升级收益系数

`UpgradeGain = Lv2Power / Lv1Power`

建议区间：`1.20 ~ 1.35`

### 地图消耗卡可用性

`MapConsumableUsable = !UsedOnMap(cardId, mapId)`

- 当 `MapConsumableUsable = false` 时，卡牌在当前地图不可再打出。

### 负面收益评分

`CurseValue = (PenaltyScore × -1) + (SynergyBonus + ConversionGain)`

用于平衡：确保负面流不是无脑最优。

### 施咒覆盖率

`DebuffCoverage = DebuffCardsPlayed / TotalTurns`

目标建议：0.35~0.55。

### X费卡强度约束

`XCostEfficiency = CardEffect(X) / X`

- X=0时允许最低基础效果（不可为0效果）
- 建议线性效率：`Effect(X) / Effect(1) ∈ [0.85X, 1.15X]`（允许轻微非线性收益）
- X的最大合理值：每回合行动点上限（默认4），X超过上限自动截断

### 固定起始卡数量约束

`InnateCountInDeck ≤ 3`

- 超过3张时，战斗开始随机保证3张固定起始卡入手
- 初始手牌剩余位置（手牌上限-固定起始数量）正常从牌库顶摸牌

### 常驻卡数量约束

`RetainedCardsPerTurn ≤ 3`

- 回合结束时若已常驻卡超过3张，自动丢弃最先常驻的卡（先进先出）

### 运筹卡累积收益上限

为防止无限叠加，运筹卡的被动效果单局上限：

`PassiveAccumulation ≤ PassiveCapPerCard`（具体上限在各卡面文本中注明）

## Edge Cases

1. 负面卡被复制：复制体保留”负面”标签。
2. 敌方免疫某类负面：施加失败但触发”施咒尝试”类被动。
3. 同回合既净化（移出卡组）又利用负面：先移出卡组后计算”持有量”收益。
4. 负面卡来源不明：日志需标明来源（事件/敌方/自加）。
5. 升级卡被复制：复制体继承当前等级，但不额外触发升级次数。
6. 使用后移除卡被回收效果影响：若文本未明确”可回收移除卡”，则不可被回收。
7. 消耗卡在同图多战斗中：首次成功打出后锁定，直到切换下一张地图才解锁。
8. **临时 + 常驻同时存在于一张卡**：不允许（互斥属性，设计时不得同时标记）。
9. **X费卡行动点为0时打出**：X=0，效果以最低档结算（不为空效果）。
10. **固定起始卡数量超过初始手牌上限（>5）**：随机取5张固定起始卡入手，剩余放回牌库。
11. **运筹卡被对手效果强制弃置（如敌方”手牌弃置”类技能）**：被动效果立即终止，卡进入弃牌堆，不可触发”被丢弃时”效果（除非卡面文本明确注明）。
12. **图鉴卡尚未解锁时出现在战斗奖励池**：系统检查玩家图鉴解锁状态，未解锁的卡不出现在奖励备选池中（战役内奖励池仅包含已解锁图鉴卡）。
13. **十三州专属卡在非对应州地图中**：专属卡只能在对应州地图通关后加入图鉴；但一旦加入图鉴，玩家用任意武将打任意地图时均可在奖励池中获得。

## Acceptance Criteria

1. 卡池总量首版目标**≥460张**（初始205 + 待开启110 + 十三州143），扩展版目标≥500张。
2. 负面卡数量**≥45张**，分布在初始通用与待开启通用两层中。
3. 具备”施咒敌方”与”负面转收益”两套可构筑框架，各含至少10张功能卡。
4. 负面卡仍可被商店/技能/特效净化，规则一致。
5. 每种卡牌均支持且仅支持一次升级（Lv1→Lv2）。
6. “使用后移除”卡在战斗内移除、局结束回卡组。
7. “消耗卡”在单张地图内仅可成功使用一次，跨地图重置。
8. **固定起始卡**：战斗开始时保证出现在起始手牌；若≥3张固定起始卡，随机取3张。
9. **常驻卡**：回合结束时自动常驻在手牌，不可选；超过3张时自动丢弃最老的常驻卡。
10. **临时卡**：回合结束若仍在手牌则自动移除，战斗结束也不加入卡组；不可被”常驻”属性阻止。
11. **运筹卡**：在手牌时被动效果持续生效；从手牌离开（打出/被弃置）后效果立即终止。
12. **X费卡**：投入X=0时产生最低档效果，不产生空效果；X值等于打出时剩余所有行动点，不可超出。
13. **图鉴解锁**：已解锁的图鉴卡不出现在奖励池；十三州专属卡在对应州通关后加入图鉴，之后全武将全地图可获得。
14. **卡牌特殊属性显示**：固定起始/常驻/移除/临时/运筹/X费在卡面有明确视觉标识（颜色标签或角标）。
