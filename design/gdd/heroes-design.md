# 武将设计文档（Heroes Design）

*Updated: 2026-03-27*
*Status: Draft v3（5张地图/武将 + 每图3幕）*

---

## Overview

本文件定义武将系统：
- 阵营：魏、蜀、吴、群雄
- 每阵营 5~6 名武将（本版22名）
- 每位武将包含：基础值、兵种倾向、**被动技能（仅1个）**、专属卡数量（12~18）、生涯地图（每武将5张地图、每图3幕、每幕10+节点）

> 已取消“武将主动技能”，以简化操作负担。

---

## Player Fantasy

“换武将=换流派”。
玩家通过不同武将被动与专属卡组，形成稳定流派分化。

---

## Detailed Rules

### 1) 武将通用基础值模板

- 生命值：65~95
- 初始手牌：5
- 每回合费用：3~4
- 军势容量：9~12
- 将令充能：默认每回合+1（上限2）

### 2) 兵种倾向

兵种：步兵 / 枪兵 / 骑兵 / 弓兵 / 谋士

每位武将定义：主修2项 + 次修1项。

### 3) 技能结构（已简化）

- 每位武将仅保留 **1个被动技能**。
- 战斗深度由：被动 + 专属卡 + 将令 + 地形 + 兵种协同构成。

### 4) 专属卡组

- 每位武将 12~18 张专属卡。
- 专属卡不进入常规卡池。

### 5) 生涯地图线（已调整）

- 每位武将生涯固定 **5张地图**（后续可扩展到6+）。
- 每张地图固定 **3幕结构**。
- 每幕至少 **10个节点**（可更多）。

地图命名建议：
- M1 起势图
- M2 扩张图
- M3 转折图
- M4 决战前图
- M5 终局图

- 基础：HP74 / 费4 / 容9

- 地图结构：5图 × 3幕 × 每幕10+节点
- 地图主题：魏廷辅政 / 祁山对峙 / 辽东征伐 / 淮南平乱 / 高平陵变

- 专属卡：13
- 地图结构：5图 × 3幕 × 每幕10+节点
- 地图主题：追随孟德 / 宛城死战 / 护驾突围 / 先锋破阵 / 忠勇余烈

- 基础：HP70 / 费4 / 容9
- 倾向：谋、步（主）/ 弓（次）

  - 每3层诡谋：本回合下一张战术卡-1费并+效果。
- 专属卡：16
- 地图结构：5图 × 3幕 × 每幕10+节点
- 地图主题：西凉从事 / 长安权谋 / 献策曹营 / 离间诸侯 / 晚局定谋

## Formulas

`EffectiveHP = BaseHP × (1 + ArmorRate + DamageReductionRate)`

`OrdersUsable ≈ BaseOrderGain + TraitBonus + ItemBonus`

---

## Edge Cases

1. 司马懿负面卡不足阈值：按当前分段即时生效，不补偿。
2. 贾诩同一动作施加多种负面：每种新负面各计1次。
3. 任意地图幕节点低于10：视为配置错误，编辑器不允许保存。
4. 武将无主动技能后，所有“主动触发描述”必须迁移至卡牌/将令。

## Tuning Knobs

- 生命：65~95
- 容量：9~12
- 专属卡：12~18
- 负面阈值段：2/5/8/9+
- 地图数量/武将：5（后续可6+）
- 每图幕数：3
- 每幕节点下限：10

---

## Acceptance Criteria

1. 每位武将仅1个被动技能，无主动技能。
2. 每阵营5~6武将，四阵营总数≥20。
3. 每位武将生涯固定5张地图，每张地图固定3幕。
4. 每幕节点数≥10。
5. 存在“自吃负面流”（司马懿）与“施加负面流”（贾诩）两类玩法。
