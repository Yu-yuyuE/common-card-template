# 武将设计文档（Heroes Design）

*Updated: 2026-03-28*
*Status: Draft v4（被动技能优化：司马懿/典韦/袁绍/吕蒙）*

---

## Overview

本文件定义武将系统：
- 阵营：魏、蜀、吴、群雄
- 每阵营 5~6 名武将（本版22名）
- 每位武将包含：基础值、兵种倾向、**被动技能（仅1个）**、专属卡数量（12~18）、生涯地图（每武将5张地图、每图3幕、每幕10+节点）

> 已取消“武将主动技能”，以简化操作负担。

---

## Player Fantasy

“换武将=换流派”。
玩家通过不同武将被动与专属卡组，形成稳定流派分化。

---

## Detailed Rules

### 1) 武将通用基础值模板

- 生命值：65~95
- 初始手牌：5
- 每回合费用：3~4
- 统御容量：9~12（用于附属兵种编制）

### 2) 兵种倾向

兵种：步兵 / 枪兵 / 骑兵 / 弓兵 / 谋士

每位武将定义：主修2项 + 次修1项。

### 3) 技能结构（已简化）

- 每位武将仅保留 **1个被动技能**。
- 战斗深度由：被动 + 专属卡 + 地形 + 附属兵种协同构成。

### 4) 专属卡组

- 每位武将 **≤12张**专属卡（司马懿≤15张，含初始2张韬晦负面卡）。
- 专属卡不进入常规卡池。

### 5) 生涯地图线（已调整）

- 每位武将生涯固定 **5张地图**（后续可扩展到6+）。
- 每张地图固定 **3幕结构**。
- 每幕至少 **10个节点**（可更多）。

地图命名建议：
- M1 起势图
- M2 扩张图
- M3 转折图
- M4 决战前图
- M5 终局图

## 魏（6）

- 地图结构：5图 × 3幕 × 每幕10+节点
- 地图主题：濮阳鏖战 / 徐州攻守 / 官渡侧翼 / 汉中对峙 / 魏境镇守

### 3) 张辽
- 基础：HP78 / 费4 / 容10
- 倾向：骑、步（主）/ 弓（次）
- 被动：威震逍遥津（骑兵首击强化）

- 地图主题：并州历练 / 下邳突击 / 合肥防线 / 逍遥津扬名 / 江淮镇压

### 4) 司马懿（负面经营流）
- 基础：HP74 / 费4 / 容9
- 倾向：谋、弓（主）/ 步（次）
- 被动：**隐忍蓄势**
  - 初始专属卡组含2张”韬晦”负面卡（常驻牌库型：在库→最大生命-3，整理卡组才消除）
  - **丢弃负面卡触发**分段奖励：
    - 牌库负面 0~2张：无增益
    - 牌库负面 3~4张：每丢弃1张负面卡，本回合+1行动点（每回合最多+2）
    - 牌库负面 5~6张：每丢弃1张负面卡，立即抽1张牌
    - 牌库负面 7+张：所有手牌费用永久-1（本战斗内）
- 专属卡：**15**（含2张初始韬晦）
- 地图结构：5图 × 3幕 × 每幕10+节点
- 地图主题：魏廷辅政 / 祁山对峙 / 辽东征伐 / 淮南平乱 / 高平陵变

### 5) 典韦（以伤换攻流）
- 基础：HP95 / 费3 / 容9
- 倾向：步、枪（主）/ 骑（次）
- 被动：**恶来**
  - 典韦**无法获得任何护甲**
  - 每次受到伤害后，本回合内下一张近战攻击卡额外造成等同于**所受伤害值**的附加伤害
  - 每次近战攻击命中敌方：回复**3**点生命
  - 生命低于40%时：近战攻击必定附带"破甲1层"
- 专属卡：**12**
- 地图结构：5图 × 3幕 × 每幕10+节点
- 地图主题：追随孟德 / 宛城死战 / 护驾突围 / 先锋破阵 / 忠勇余烈

### 6) 贾诩（施加负面流）
- 基础：HP70 / 费4 / 容9
- 倾向：谋、步（主）/ 弓（次）
- 被动：**毒士**
  - 每次给敌方施加1种新负面**状态**，获得**1层诡谋**
  - 消耗**2层**诡谋：本回合下一张卡费用-1（无下限限制，叠加效果可使卡费为0）
- 专属卡：**12**
- 地图结构：5图 × 3幕 × 每幕10+节点
- 地图主题：西凉从事 / 长安权谋 / 献策曹营 / 离间诸侯 / 晚局定谋

## Formulas

`EffectiveHP = BaseHP × (1 + ArmorRate + DamageReductionRate)`

`OrdersUsable ≈ BaseOrderGain + TraitBonus + ItemBonus`

---

## Edge Cases

1. 司马懿负面卡不足阈值：按当前分段即时生效，不补偿。
2. 贾诩同一动作施加多种负面：每种新负面各计1次。
3. 任意地图幕节点低于10：视为配置错误，编辑器不允许保存。
4. 武将无主动技能后，所有”主动触发描述”必须迁移至卡牌。

## Tuning Knobs

- 生命：65~95
- 统御容量：9~12（附属兵种编制预算）
- 专属卡：12~18
- 负面阈值段：2/5/8/9+
- 地图数量/武将：5（后续可6+）
- 每图幕数：3
- 每幕节点下限：10

---

## Acceptance Criteria

1. 每位武将仅1个被动技能，无主动技能。
2. 每阵营5~6武将，四阵营总数≥20。
3. 每位武将生涯固定5张地图，每张地图固定3幕。
4. 每幕节点数≥10。
5. 存在“自吃负面流”（司马懿）与“施加负面流”（贾诩）两类玩法。
