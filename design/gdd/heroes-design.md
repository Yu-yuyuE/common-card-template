# 武将设计文档（Heroes Design）

_Updated: 2026-03-30_
_Status: Draft v5（HP全武将+10，被动技能同步CSV）_

---

## Overview

本文件定义武将系统：

- 阵营：魏、蜀、吴、群雄
- 每阵营 5~6 名武将（本版22名）
- 每位武将包含：基础值、兵种倾向、**被动技能（仅1个）**、专属卡数量（12~18）、生涯地图（每武将5张地图、每图3幕、每幕10+节点）

> 已取消“武将主动技能”，以简化操作负担。

---

## Player Fantasy

“换武将=换流派”。
玩家通过不同武将被动与专属卡组，形成稳定流派分化。

---

## Detailed Rules

### 1) 武将通用基础值模板

- 生命值：40~60
- 初始手牌：5
- 每回合费用：3~4
- 统帅：3~6（可携带兵种卡上限）

### 2) 兵种倾向

兵种卡类型：步兵 / 骑兵 / 弓兵 / 谋士 / 盾兵

每位武将定义：主修2项 + 次修1项。

> 兵种倾向影响军营节点的兵种卡出现权重：主修兵种卡的出现概率更高。倾向不限制武将可选的兵种卡类型，仅作为构筑引导。

### 3) 技能结构（已简化）

- 每位武将仅保留 **1个被动技能**。
- 战斗深度由：被动 + 专属卡 + 地形 + 兵种卡协同构成。

### 4) 专属卡组

- 每位武将 **≤12张**专属卡（司马懿≤15张，含初始2张韬晦负面卡）。
- 专属卡不进入常规卡池。

### 5) 生涯地图线（已调整）

- 每位武将生涯固定 **5张地图**（后续可扩展到6+）。
- 每张地图固定 **3幕结构**。
- 每幕至少 **10个节点**（可更多）。

地图命名建议：

- M1 起势图
- M2 扩张图
- M3 转折图
- M4 决战前图
- M5 终局图

## Formulas

`EffectiveHP = BaseHP × (1 + ArmorRate + DamageReductionRate)`

`OrdersUsable ≈ BaseOrderGain + TraitBonus + ItemBonus`

---

## Edge Cases

1. 司马懿负面卡不足阈值：按当前分段即时生效，不补偿。
2. 贾诩同一动作施加多种负面：每种新负面各计1次。
3. 任意地图幕节点低于10：视为配置错误，编辑器不允许保存。
4. 武将无主动技能后，所有”主动触发描述”必须迁移至卡牌。

## Tuning Knobs

- 生命：40~60
- 统帅：3~6（可携带兵种卡上限）
- 专属卡：12~18
- 负面阈值段：2/5/8/9+
- 地图数量/武将：5（后续可6+）
- 每图幕数：3
- 每幕节点下限：10

---

## Acceptance Criteria

1. 每位武将仅1个被动技能，无主动技能。
2. 每阵营5~6武将，四阵营总数≥20。
3. 每位武将生涯固定5张地图，每张地图固定3幕。
4. 每幕节点数≥10。
5. 存在“自吃负面流”（司马懿）与“施加负面流”（贾诩）两类玩法。
