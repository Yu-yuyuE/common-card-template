## MVP 关键路径

能够运行可玩战斗原型所需的最少系统：

```
F2（资源管理）→ C1（状态效果）→ C2（卡牌战斗）→ C3（敌人）
                                        ↓
                                  D1（地形天气）→ D2（兵种卡）
                                        ↓
                                   D3（武将）
```

**首个可玩里程碑**：F2 + C1 + C2 + C3（4 个系统）
**完整 MVP**：F2 + C1 + C2 + C3 + D1 + D2 + D3 + D4 + M1（9 个系统）
