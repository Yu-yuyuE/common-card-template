# 游戏术语表 (Game Glossary)

本文档是游戏中所有术语的**唯一标准来源**。
策划文档（GDD）、程序代码（Code/ID）、以及 UI 显示（UI Text）必须严格遵循此处的定义。如果在开发中发现新的术语歧义，请务必将其添加到本表中。

## 状态系统 (Status System)

| 内部 ID (Code/ID) | 标准中文名 (Display Name) | 废弃/别名 (Deprecated) | 英文翻译 (EN) | 定义与作用说明 (Definition) |
| :--- | :--- | :--- | :--- | :--- |
| `STATUS_EVASION` | 闪避 | 迅捷 | Evasion | 受到攻击时有概率完全避免伤害的状态。 |

## 地形系统 (Terrain System)

| 内部 ID (Code/ID) | 标准中文名 (Display Name) | 废弃/别名 (Deprecated) | 英文翻译 (EN) | 定义与作用说明 (Definition) |
| :--- | :--- | :--- | :--- | :--- |
| `TERRAIN_PASS` | 关隘 | 通道 | Pass | 连接不同区域的狭窄地形，通常具有战略防守意义或特殊的通行限制。 |

> **使用原则：**
> 1. **代码规范**：代码中禁止使用魔法字符串，请使用大写的内部ID对应的枚举或常量。
> 2. **文档规范**：编写和修改 ADR 或 GDD 时，必须使用“标准中文名”。如果需要，可以标注英文对照。
> 3. **UI规范**：UI 层面仅允许通过翻译系统的 Key 引用，最终展示为“标准中文名”。
