# MVP 战斗原型：1v3 + 兵种卡 + 地形

## 目标
验证核心战斗循环的可行性：玩家能否通过选择兵种卡在1v3战场中获得战术优势？

## 范围
- 玩家：1个主将，5张兵种卡
- 敌人：3个固定单位（无AI复杂逻辑）
- 系统：兵种卡效果、地形修正（平原/山地/森林/水域）
- 不包含：费用系统、状态效果、天气、升级、诅咒

## 如何运行
1. 打开 Godot 4.6.1
2. 加载 `prototype-combat-mvp.tscn`
3. 使用鼠标点击手牌中的兵种卡进行出牌
4. 观察敌方单位的反应
5. 尝试不同地形下的战斗表现

## 调试提示
- 按 `F1` 显示调试信息
- 按 `R` 重置战斗
- 所有数据硬编码，不依赖生产代码
